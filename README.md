# docker-python
A minimal docker, python, uwsgi and ngix configuration. Simple random dice generator.

# Getting Started
1. Clone Repo.
2. Run docker-compose build.
3. Run docker-compose up.
4. You can now modify the app in real-time.

# About
The API uses the python bottle micro-framework. See repository [HERE](https://github.com/bottlepy/bottle)
